# Bahrain Utility Tariff Calculator

**حاسبة التعرفة الكهربائية، المائية والبترولية – البحرين**

**Done by:** Eng. Mohamed Jaber ALASHEERI

## الوصف
تطبيق تفاعلي لحساب التعرفة الكهربائية، المائية والبترولية في البحرين.  
- يدعم العربية والإنجليزية.  
- يعرض الفاتورة القديمة، الجديدة، الفرق، ونسبة الزيادة.  
- يحتوي على Visuals جذابة توضح الشرائح التي وصل إليها المستخدم.  
- Responsive تمامًا للشاشات الصغيرة والموبايل.

## الملفات
- `app.py` : الكود الرئيسي للتطبيق.  
- `fuel_prices.csv` : أسعار الوقود السابقة والجديدة.  
- `requirements.txt` : مكتبات Python المطلوبة لتشغيل التطبيق.

## طريقة التشغيل
1. افتح [Streamlit](https://share.streamlit.io).  
2. اختر المستودع وملف `app.py`.  
3. اضغط **Deploy**.  
4. التطبيق جاهز للعمل مباشرة.

## تحديث الأسعار
- أسعار الكهرباء والماء محدثة حسب القرارات الأخيرة.  
- أسعار الوقود يمكن تحديثها شهريًا في `fuel_prices.csv`.
